# kicad-viewer

Browser-based KiCad 10 project viewer. Part of the `kicad-tools` monorepo.

## Develop

```
cd kicad-viewer
npm install
npm run dev
```

## Build

```
npm run build
```

Static output lands in `build/`. Deployable to Cloudflare Pages or any static host.
